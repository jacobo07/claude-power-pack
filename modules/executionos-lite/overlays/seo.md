# Overlay: SEO/GEO

- **Local SEO:** NAP consistency across all citations. Google Business Profile optimization before content.
- **GEO (Generative Engine Optimization):** Structure content for AI consumption. Entity-rich, claim-backed, source-cited.
- **Content Strategy:** Topic clusters with pillar pages. Internal linking minimum 3 per article.
- **Technical SEO:** Core Web Vitals targets: LCP < 2.5s, INP < 200ms, CLS < 0.1. Structured data on all pages.
- **Schema Markup:** Use JSON-LD. LocalBusiness, FAQPage, HowTo, Article as applicable. Validate with Google Rich Results Test.
- **Crawl Budget:** Robots.txt + sitemap.xml maintained. No orphan pages. Canonical tags on all pages.
- **Monitoring:** Track: organic impressions, CTR, keyword positions, AI citation appearances.
- **Content Quality:** E-E-A-T signals: author bios, source citations, publish dates, update dates.
- **Automation:** Python-native pipelines only. No n8n. WordPress updates via REST API with raw content.
- **Reporting:** Data-backed recommendations only. No vanity metrics. ROI framing for client deliverables.
