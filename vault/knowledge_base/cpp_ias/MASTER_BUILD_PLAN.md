## 0. What this mission is (and is not)

The CPP-ACI Reality Audit (STOP #1) proved that building the 12 sciences would duplicate
~500k words of sealed PP work. The Owner's ruling: **do not build the duplicates — convert each
into a second-order system.** CPP-IAS is that corpus.

**The one institutional object no existing system owns.** Every PP system today governs *one
resource type* at *one level*:

| Existing governor | Governs | At the level of |
|---|---|---|
| CO-03 Dynamic Cognitive Router | model selection | one session's model calls |
| PM-01..05 Parallel Mesh | pane coordination | panes on one repo |
| GK-00..12 Graphify | knowledge location | knowledge nodes |
| D2A Engine | one duplication event | a single proposal |
| CO-01 / token_irr / recall_roi | cost & reuse | one session / one asset |
| cascade_prevention / osa | error containment | one session's errors |
| skill_router | skill selection | one prompt |

**None governs the ENSEMBLE of 70+ systems as a single organism.** CPP-IAS's institutional
object is *the system-of-systems* — the federation as a managed portfolio, ecology, economy,
immune system, and self-model. That object is genuinely unowned, and it is exactly the
"circulatory / nervous / immune / economic system" the Owner named. **This is the non-duplication
boundary for the entire family:** every IAS dataset governs the ensemble; the moment a dataset
would govern a single resource type, it is duplicating a parent and is rejected.

## 2. Institutional Advantage Score (selection heuristic — qualitative, not false-precise)

```
Institutional Advantage Score  ≈
    white_space × systems_strengthened × expected_reuse × cross_project_applicability
        × compounding_effect × model_independence
    ─────────────────────────────────────────────────────────────────────────────────
    complexity_introduced × overlap_with_parents × maintenance_cost
```

Applied qualitatively (D2A's numeric-benchmark discipline, no adjectives-without-mechanism).
A dataset is admitted only if it **strengthens ≥2 existing systems** and its institutional
object is **distinct from every parent** (proven in the ledger). Priority datasets strengthen 5–20.
