# CPP-IAS — SYSTEM-STRENGTHENING MATRIX

> Every IAS dataset must strengthen ≥2 existing systems; priority datasets 5–20.
> Format: `new system → existing systems strengthened → mechanism → expected gain → evidence required`.
> "Strengthen" = the parent produces more value, makes better decisions, or costs less *because
> the IAS system exists* — never "the IAS system replaces it".

## The matrix law
A dataset may not seal until its row here is filled AND its "evidence required" column names a
concrete, checkable artifact. A strengthening claim without required evidence is a hypothesis,
marked HYPOTHESIZED, and may not be stated as fact in the dataset.
